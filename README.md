# FoodFits

A simple web app for healthy eating.

## Getting Started

1. Open `index.html` in your browser to view the app.
2. Edit `style.css` to customize the appearance.
3. Add functionality in `script.js`.

## Development

Use the Live Server extension in VS Code to run the app with live reload:
- Right-click on `index.html` and select "Open with Live Server".

## Features

- Basic HTML structure
- Styled with CSS
- JavaScript ready for enhancements
